Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito (não semântico) usando a tag <b> .

Nessa frase temos um termo em destaque (semântico) usando <strong>;.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>; (não semântica)

Nessa frase, temos um termo em ênfase (semântico) usando a tag <em>.

Principais Formatações

Podemos criar um texto marcado usando a tag <mark>.

Também podemos marcar um texto usando o comando 'mark style'.

Texto Grande e Pequeno

Estamos criando um texto grande (em desuso atualmente) e um texto pequeno nesse parágrafo usando <big> e <small>, respectivamente.

Texto Deletado

Podemos marcar um texto como excluído utilizando a tag <del> para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido utilizando a tag <ins> (semântico) para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado utilizando a tag <u> (não semântico).

Texto Sobrescrito e Subscrito

O SOBRESCRITO é usado para inserir fórmulas matemáticas, por exemplo uma potenciação, utilizando a tag <sup>. X2

O SUBSCRITO é utilizado para fórmulas químicas utilizando a tag <sub>. H20
